# Mobile App

This is the Expo Router app for the golf GPS and performance product.

## Commands

```bash
npm install
npx expo start
npm run lint
npm run typecheck
```

## EAS Builds

```bash
npx eas build --platform ios --profile preview
npx eas build --platform android --profile preview
```

## Environment

Create a local `.env` file from `.env.example` before wiring Supabase.
