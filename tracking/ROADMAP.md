# Delivery Roadmap

## Sprint 0: Expo Foundation

Goal:

- align docs to React Native and Expo
- scaffold the mobile app
- add core dependencies
- define the initial domain modules

## Sprint 1: Course Data and GPS

Goal:

- playable course model
- manual or imported course catalog
- location permissions
- front, center, and back yardage calculations

## Sprint 2: Round Engine and Scoring

Goal:

- start round flow
- current hole state
- score entry
- hole progression
- local round persistence

## Sprint 3: History, Stats, and Sync

Goal:

- round history
- round detail screen
- basic stats dashboard
- Supabase sync
- offline recovery

## Sprint 4: Build Hardening and Release Prep

Goal:

- EAS build profiles
- device testing
- GPS accuracy checks
- field validation
- beta release checklist
