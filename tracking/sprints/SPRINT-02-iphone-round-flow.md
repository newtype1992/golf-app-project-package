# Sprint 2: iPhone Active Round MVP

Status: Not started

## Goal

Make the iPhone app capable of starting and managing a round end to end.

## Scope

- [ ] course list and course selection
- [ ] start-round flow
- [ ] active-hole yardage screen
- [ ] phone score entry for strokes, putts, fairway hit, GIR, and penalties
- [ ] next-hole progression
- [ ] save and resume active round

## Definition of done

- the user can start a round on iPhone
- hole-by-hole scores save reliably
- the current hole is always obvious
